Chào các bạn! Mình tên là Phạm Hồng Triệu, hiện là sinh viên chuyên ngành Công nghệ thông tin. Ngay từ những ngày đầu bước chân vào giảng đường đại học, mình đã bị cuốn hút bởi lập trình và việc giải quyết các bài toán thông qua dòng code.

Mục tiêu nghề nghiệp của mình là trở thành một Software Engineer chuyên nghiệp. Mình tập trung nghiên cứu lập trình web hiện đại (HTML/CSS/JS), các ngôn ngữ lập trình phổ biến như Python và Java, cùng với các hệ quản trị cơ sở dữ liệu như SQL Server.

Mình thích tạo ra các sản phẩm thực tiễn và chia sẻ chúng trên GitHub để học hỏi kinh nghiệm cũng như cải thiện chất lượng code từ cộng đồng.
